encryption.c
	this file prompts the user for a key and file name.
	it then waits on user input. The input is encrypted
	and written to the file specified.

decryptio.c
	this file prompts the user for a key and file name.
	it then decrypts the file contents and displays them
	to the screen.

a make file is included in the submission the commands are

make encryption
	makes Encryption executable

make decryption
	makes Dencryption executable

make clean
	removes executables
